# アプリケーション名
ShoppingDays

# アプリケーション概要
カレンダースケジュールを土台として買い物リストを共有することで、買うものの情報共有ができて買い忘れ防止もできる。

# 利用方法	このアプリケーションの利用方法を記載。説明が長い場合は、箇条書きでリスト化すること。
・トップページからユーザー新規登録画面に移り、ユーザー新規登録を行う。※新規登録時はログインされて、カレンダーページに移動する。
・ユーザーは「ログアウト」リンクよりログアウトできる。
・ユーザーは「ログイン」リンクよりログインできる。
・カレンダーの＋マークをクリックして、買い物リストを作成する。
・買い物リストの詳細画面でアイテムを作成（追加）・編集・削除する。
・画面右上の「新しい予定を作成」ボタンにてプランを作成する。
・プランの詳細画面でプランを編集・削除する。

# アプリケーションを作成した背景	このアプリケーションを通じて、どのような人の、どのような課題を解決しようとしているのかを記載。
日常生活において、スーパーやコンビニに買い物に行ったときに、
家族に頼まれた日用品や食料品を頻繁に買い忘れることがあり、買い忘れを防ぐことができないかと考えました。
自分と家族、世の中の主婦や主夫のような毎日買い物をする人が、買い忘れを防止できるように、簡単かつ手軽に操作できるアプリの作成を行いました。
